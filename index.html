About Me

Hi, my name's Josue Guerrero and I'm a front-end developer.

The First time I got drawn into the world of computers was when my dad brought home a brand new PC with Windows 98 OS, at the time I thought it was the best thing in the world. I would play MS-DOS games in it and think how this square box could run games, and how to make them.

I've decided that I want to be a Web Developer and so my journey begins. I'm a young Venezuelan man, married and with alot of hunger for new knowledge.

I've currently finished some HTML5 and CSS3 courses and moving onward to JavaScript ones, so check back anytime and see if I've made any changes.

This portfolio has been uploaded to gitHub so if you want to use it its all for the taking, although I would love to see what you made with it in end, so if you wish I'd love to hear from you.
